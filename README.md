# 熊海斌个人学术主页

这是一个可直接部署到 GitHub Pages 的静态个人学术简历网站，包含中文和英文切换、个人照片、研究方向、教育背景、技能证书和部分论文。

## 文件结构

```txt
.
├── index.html
├── styles.css
├── script.js
└── assets/
    └── profile.jpg
```

## GitHub Pages 发布

1. 新建仓库，推荐命名为 `你的GitHub用户名.github.io`。
2. 上传本目录中的 `index.html`、`styles.css`、`script.js` 和 `assets/profile.jpg`。
3. 在仓库 `Settings` -> `Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
4. 保存后等待发布完成。
